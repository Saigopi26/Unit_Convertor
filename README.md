# Unit Converter Web App

## Project URL
[Unit_Converter](https://roadmap.sh/projects/unit-converter)

## Description
This project is a simple **Unit Converter Web Application** that allows users to convert values between different units of measurement.

The application provides separate sections for different unit types such as **Length**, **Weight**, and **Temperature**.  
Users can enter a value, select the units to convert from and to, and instantly view the converted result.

## Features
- Input a value to convert
- Select units to convert from and to
- View the converted output instantly
- Supports multiple categories of units
- Simple and user-friendly interface

## Supported Units

### Length
- Millimeter
- Centimeter
- Meter
- Kilometer
- Inch
- Foot
- Yard
- Mile

### Weight
- Milligram
- Gram
- Kilogram
- Ounce
- Pound

### Temperature
- Celsius
- Fahrenheit
- Kelvin

## How the Application Works
1. The user selects a measurement category (Length, Weight, or Temperature).
2. The user enters the value to be converted.
3. The user selects the source unit and target unit.
4. The application calculates and displays the converted value.

## Technologies Used
- Python
- Streamlit (for building the web interface)
